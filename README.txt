|                                |
|    INSTALAR BRISTOL MANAGER    |
|      Guía de instalación:      |
|                                |
----------------------------------

Para que la instalación funcione correctamente
debe seguir atentamente los siguientes pasos:

1. Esta carpeta debe llamarse exactamente "bristolManager"

2. Entra en la carpeta "deploy"

3. Ejecuta el archivo "install.bat" (su icono será algo similar
a unas tuercas o engranajes) haciendo doble click

4. Se abrirá una consola, sigue atentamente los pasos
que aparezcan en dicha terminal
